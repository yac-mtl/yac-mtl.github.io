Sublime Text est un autre éditeur HTML. Vous pouvez l’utiliser gratuitement, cependant, si vous désirez pouvoir disposer de toutes les fonctionnalités, il vous sera nécessaire de souscrire à une licence payante. Ce logiciel vous permet d’ajouter des fonctionnalités sous forme de plugin, créé par les membres de la communauté. Vous pouvez aussi, si vous le désirez, créer vos propres plugins. Sublime Text prend en charge les API Python, ce qui permet d’étendre les fonctionnalités. De plus, il vous permet d’éditer simultanément plusieurs Zones de codes. Ce logiciel est disponible aussi bien sous Windows que Linux et OS X.
