NotePad++ est un éditeur spécialement conçu pour Windows. Cependant, il vous est possible de l’utiliser sous Linux par le biais de Wine. Cet éditeur HTML est libre d’utilisation et totalement gratuit. De plus ce dernier est super léger et dispose d’une version mobile.

Les avantages de cet éditeur sont les suivants:

  1. Le logiciel NotePad++ est totalement gratuit.
  2. Il est extension grâce à l’ajout de plugins créé par la communauté et qu’il vous est possible de trouver sur GitHub.
  3. Si vous êtes un développeur expérimenté, il vous est possible de personnaliser les fonctionnalités de ce logiciel..
