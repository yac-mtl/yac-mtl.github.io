Introduit pour la première fois en 2014, il est rapidement devenu l’un des meilleurs éditeurs HTML disponibles. L’objectif de l’éditeur ATOM est d’être accueillant pour les débutants mais aussi complet pour les développeurs Web expérimentés. De plus l’éditeur ATOM est largement personnalisable avec plus de 7500 paquets installables. De plus, les thèmes proposés sont tout aussi personnalisables avec cet éditeur. Plus de 2500 thèmes sont disponibles et peuvent être téléchargés.

Les avantages de cet éditeur sont les suivants:

  1. Personnalisable. Il vous est très facile de personnaliser l’éditeur, que ce soit pour en modifier l’apparence, mais aussi pour y ajouter des fonctionnalités.
  2. Le mode Dev. Il vous est possible d’expérimenter l’ajout de fonctionnalités au système principal de l’éditeur.
  3. Git et GitHub. Vous pouvez intégrer facilement Git et GitHub à votre éditeur.
